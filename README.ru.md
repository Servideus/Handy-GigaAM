# Handy-GigaAM (русская документация)

Этот репозиторий является форком оригинального проекта [Handy](https://github.com/cjpais/Handy), ориентированным на русскоязычных пользователей.

## Что изменено в форке

- Добавлены модели **GigaAM v3 e2e-CTC**:
  - `GigaAM v3 e2e-CTC (сжатая)` (int8)
  - `GigaAM v3 e2e-CTC (полная)`
- Улучшена документация и сценарии релиза для русского использования.
- Сборка и релизы остаются полностью офлайн-ориентированными по распознаванию речи (после загрузки модели).

## Быстрый старт

1. Скачайте релиз из раздела **Releases** этого форка:
   - <https://github.com/Servideus/Handy-GigaAM/releases>
2. Установите приложение и выдайте разрешения (микрофон и доступность).
3. В настройках выберите нужную модель (для русского языка рекомендуется GigaAM).
4. Настройте горячую клавишу и начните диктовку.

## Важно для Windows

Если сборка не подписана, предупреждение SmartScreen является ожидаемым.
В этом случае выберите **Подробнее** -> **Выполнить в любом случае**.
Скачивайте установщики только из GitHub Releases этого репозитория.

## Подпись релизов и безопасность

- Политика подписи и процесс релизов: [CODE_SIGNING_POLICY.md](CODE_SIGNING_POLICY.md)
- Подпись Windows-артефактов через SignPath включается опционально в CI и не обязательна для сборки.
